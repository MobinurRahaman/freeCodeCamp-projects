Introduction

Dart is a client-optimized programming language developed by Google for building fast apps on any platform. Dart is an object-oriented language with C-style syntax and supports strong typing, interfaces, mixins, abstract classes, and more. Dart also includes features like async/await, futures, and isolates for writing concurrent and asynchronous code. Dart can be compiled to native code or JavaScript, making it a versatile language for building a wide range of applications.

Some key features of Dart include:

  • Strong typing and null safety
  • Rich core libraries
  • Support for asynchronous programming
  • Native performance on multiple platforms
  • Easy integration with JavaScript

Installing Dart

To install Dart, follow the instructions provided on the Dart website: https://dart.dev/get-dart.

Basic Syntax

A simple Dart program starts with the main function:


void main() {
  print('Hello, World!');
}

Comments in Dart can be single-line or multi-line:


// This is a single-line comment

/*
This is a
multi-line comment
*/

Comments

Comments in Dart are used to explain the code and make it more readable. A comment in Dart starts with // for a single-line comment or /* */ for a multi-line comment. Here is an example:


// This is a single-line comment in 
/*
  This is a multi-line comment in Dart
  It can span multiple lines
*/

  // This is a comment inside a function
  print('Hello, world!'); // This is also a comment
}

Variables and Data Types

Dart has built-in support for the following data types:

  • Numbers (int and double)
  • Strings
  • Booleans
  • Lists
  • Maps

Here's an example of declaring variables in Dart:


int age = 30;
double height = 5.9;
String name = 'John Doe';
bool isStudent = false;
List<int> numbers = [1, 2, 3, 4, 5];
Map<String, String> userInfo = {'name': 'John', 'email': 'john@example.com'};

Control Flow Statements

Dart supports common control flow statements like if-else, for loop, and while loop:


// If-else statement
if (age > 18) {
  print('You are an adult');
} else {
  print('You are not an adult');
}

// For loop
for (int i = 0; i < 5; i++) {
  print(i);
}

// While loop
int count = 0;
while (count < 5) {
  print(count);
  count++;
}

Functions

Functions in Dart can be defined with the `void` keyword or with a return type:


void greet() {
  print('Hello, World!');
}

int add(int a, int b) {
  return a + b;
}

void main() {
  greet();
  int sum = add(5, 3);
  print('Sum: $sum');
}

Classes

In Dart, a class is a blueprint for creating objects, which are instances of that class. A class defines the attributes and methods that an object can have, as well as its behavior and functionality.

To define a class in Dart, use the class keyword followed by the class name, and then the class body enclosed in curly braces:


/// The `Person` class represents a person with a name and age.
class Person {
  /// The name of the person.
  String name;

  /// The age of the person.
  int age;

  /// Creates a new `Person` instance with the given `name` and `age`.
  Person(this.name, this.age);

  /// Prints a greeting message to the console.
  void sayHello() {
    print('Hello, my name is $name and I am $age years old.');
  }
}

void main() {
  final person = Person('John', 25);
  person.sayHello();
}
